# ✈️ Airline Flight Cancellation Analysis — U.S. Domestic Flights (2015)

A data analytics project exploring flight operations across U.S. domestic airlines in 2015. This analysis investigates delays, cancellations, traffic patterns, airline performance, and operational trends using approximately 2 million flight records.

The goal is to uncover actionable insights that help understand factors affecting flight reliability and passenger experience.

---

## 📊 Project Overview

Air transportation performance impacts millions of travelers daily. Delays and cancellations create operational inefficiencies and reduce customer satisfaction.

This project analyzes:

- Monthly flight trends
- Delay patterns
- Cancellation causes
- Airline performance
- Airport traffic concentration
- Day-of-week operational behavior
- Overall flight outcomes

The project was designed as a business-focused stakeholder/shareholder report.

---

## Dataset Summary

- Year: 2015
- Scope: U.S Domestic Flights
- Total Flights: ~2 Million
- Delayed Flights: ~790,000
- Cancelled Flights: ~29,000

---

## Tools Used
- Power BI 
- Excel / CSV processing
- Data visualization techniques
- Exploratory Data Analysis (EDA)

---

## Key Findings

### Monthly Trends

- Summer months (June–August) experienced the highest traffic volume and delays.
- February recorded the highest cancellations:
  - 6,432 cancelled flights
- September had the lowest delay and cancellation rates.


---

### Airline Delay Performance

Best Performing:

- US Airways Inc → 30.5% delayed
- Atlantic Southeast Airlines → 31.4% delayed

Worst Performing:

- United Air Lines Inc → 52.6% delayed
- Southwest Airlines Co. → 51.2% delayed

---

### Cancellation Analysis

Weather emerged as the dominant cancellation factor:

- 57.3% of all cancellations were weather-related

This suggests opportunities for:

- predictive weather systems
- flexible scheduling
- proactive passenger communication
